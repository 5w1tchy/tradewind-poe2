# Fixtures — clipboard sample corpus

How to collect:

1. In game, hover an item and press **Ctrl+Alt+C** (advanced copy — includes mod tiers/ranges).
2. Open `inbox.txt`, paste at the bottom.
3. Put a line with just `=====` after each item so they're separated.
4. Repeat. Don't worry about order, duplicates, or formatting — I'll split, dedupe, and sort them into per-category fixture files later.

What to grab (variety beats quantity):

- Rares with hybrid mods, desecrated/fractured mods, corrupted ones
- Items with runes/soul cores socketed
- Uniques (cheap and expensive)
- Waystones, tablets, jewels, charms, flasks
- Weapons (the DPS math cases: bows, crossbows, quarterstaves, maces)
- Gems/uncut gems
- Anything that looks weird or league-specific

~30–50 items is a great corpus. More never hurts.
